##Summary
This is a code that simplifies and automates some of the OCR review process. Notable features include
1. Cleans and formats XML files into text entries.
2. Can load and save reviews from XML files
3. Exports 2 CSVs - an errors CSV that records where errors were found, and an error frequency CSV that records the occurances of errors for spotting systematic issues.
4. Easy buttons with common issues that automatically formats CSV entries

##Using/Navigating
The main application is saved in /dist.
####THE DATA FOLDER AND ITS CONTENTS MUST BE IN THE SAME DIRECTORY AS THE APPLICATION OR CODE TO RUN
\n The data folder has 3 files: "lastsession.json", "errors.csv", and "errorFreq.csv".
The VS code workspace and venv are also provided, but the data folder will have to be pulled from dist.
