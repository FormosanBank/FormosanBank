These audio files were collected by Prof. Apay Tang, a member of the Truku tribe and a prominent perservationist and revitalizationist in the community. While permanently archived at Paradisec, Prof. Tang graciously agreed to include the audio in FormosaBank.

None of the audio are transcribed. There is slightly more meta-data at Paradisec, but not much.

## Project Structure

- **Audio/Truku/**: WAV audio files recorded by Prof. Apay Tang
- **Metadata/**: RO-Crate JSON metadata files from Paradisec (one per collection item)
- **XML/Truku/**: FormosanBank XML files generated by `make_xml.py` (one per WAV file)
- `make_xml.py`: Generates the XML files from the audio files and metadata
- `upload_hf_datasets.sh`: Uploads `Audio/` to the HuggingFace dataset repo
- `download_audio_data.sh`: Downloads `Audio/` from the HuggingFace dataset repo

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/FormosanBank/Formosan-Tang-Recordings-of-Taroko.git
   cd Formosan-Tang-Recordings-of-Taroko
   ```

2. Install required Python packages (best done in a virtual environment):
   ```bash
   pip install -r requirements.txt
   ```
   Note: all dependencies are Python standard library. Python >= 3.9 is required.

3. Obtain the raw audio:

For strict reproduction, you'd get this from Paradisec. But that requires a lot of paperwork and would cause unnecessary work for others. You can download directly from FormosanBank:

```bash
    ./download_audio_data.sh
```

## Usage

Generate the XML files from the audio files and metadata:

```bash
python make_xml.py
```

This will:
- Scan `Audio/Truku/` for WAV files
- Look up metadata for each file from the corresponding JSON in `Metadata/`
- Write one FormosanBank-format XML file per WAV into `XML/Truku/`
