

# Ortho94

The orthography released in ROC year 94

## Amis

The letter /f/ can be pronounced as /b/, /v/, or /f/, depending on dialect. For purposes of standardization, we chose /f/, which is common in the northern dialect.

The letter /d/ similarly has three pronunciations (which use non-standard characters that were hard to copy into this document). We used /d/, which is common in the northern dialect.

The chart uses a strange r-looking IPA symbol for /l/ that I could not otherwise find. From context, it appears to be for the lateral flap /ɺ̠/.
